`cg_auth_token` - Token de autenticação\n   - `cg_current_user` - Dados do usuário\n4. **Novo jogo lê `localStorage` e já está autenticado**\n\n### Implementação em Novo Jogo\n\n```javascript\n// No seu novo-jogo/script.js\n\n// 1. Verificar se já está autenticado (vindo da plataforma)\nconst platformUser = localStorage.getItem('cg_current_user');\nconst platformToken = localStorage.getItem('cg_auth_token');\n\nif (platformUser && platformToken) {\n    // Usuário já está autenticado pela plataforma\n    const userData = JSON.parse(platformUser);\n    console.log('✅ Usuário autenticado:', userData.username);\n    \n    // Pular tela de login e ir direto para o jogo\n    showScreen('main');\n    initializeGame();\n} else {\n    // Mostrar tela de login\n    showScreen('login');\n}\n\n// 2. Quando usuário faz login/registro no jogo\nasync function handleAuth(email, password) {\n    try {\n        // Chamar Supabase\n        const { data, error } = await supabase.auth.signInWithPassword({\n            email,\n            password\n        });\n        \n        if (error) throw error;\n        \n        // Salvar em localStorage (para sincronizar com plataforma)\n        localStorage.setItem('cg_auth_token', data.session.access_token);\n        localStorage.setItem('cg_current_user', JSON.stringify({\n            id: data.user.id,\n            email: data.user.email,\n            username: data.user.user_metadata?.username\n        }));\n        \n        // Inicializar jogo\n        showScreen('main');\n        initializeGame();\n    } catch (error) {\n        console.error('❌ Erro ao fazer login:', error);\n    }\n}\n```\n\n---\n\n## 💾 Sistema de Dados Global\n\n### Dados Compartilhados Entre Jogos\n\nTodos os jogos compartilham os mesmos dados do usuário no Supabase:\n\n| Campo | Tabela | Descrição |\n|-------|--------|----------|\n| `user_id` | `users` | ID único do usuário |\n| `coins` | `game_stats` | Moedas globais (ganhas em qualquer jogo) |\n| `xp` | `game_stats` | XP global (ganho em qualquer jogo) |\n| `spin_tickets` | `game_stats` | Fichas da roleta (compartilhadas) |\n| `active_theme` | `game_stats` | Tema ativo (compartilhado entre jogos) |\n| `item_id` | `shop_items` | Itens possuídos (compartilhados) |\n| `user_id` | `friends` | Amigos (compartilhados) |\n\n### Dados Específicos do Jogo\n\nCada jogo pode ter suas próprias tabelas:\n\n```javascript\n// Exemplo: Novo jogo com stats específicos\n\n// Tabela: novo_jogo_stats\n{\n    user_id: uuid,\n    level: integer,\n    score: integer,\n    games_played: integer,\n    wins: integer,\n    losses: integer,\n    best_score: integer,\n    // ... outros campos específicos do jogo\n}\n\n// Tabela: novo_jogo_achievements\n{\n    user_id: uuid,\n    achievement_id: string,\n    unlocked_at: timestamp,\n    // ... outros campos\n}\n```\n\n### Como Sincronizar Dados\n\n```javascript\n// No seu novo-jogo/supabase-sync.js\n\n/**\n * Carregar dados do usuário (global + específico do jogo)\n */\nasync function loadUserData(userId) {\n    const client = getSupabaseClient();\n    \n    try {\n        // 1. Carregar dados globais (moedas, XP, etc)\n        const { data: globalStats } = await client\n            .from('game_stats')\n            .select('*')\n            .eq('user_id', userId)\n            .single();\n        \n        // 2. Carregar dados específicos do jogo\n        const { data: gameStats } = await client\n            .from('novo_jogo_stats')\n            .select('*')\n            .eq('user_id', userId)\n            .single();\n        \n        // 3. Carregar inventário (itens compartilhados)\n        const { data: inventory } = await client\n            .from('shop_items')\n            .select('item_id, is_active')\n            .eq('user_id', userId);\n        \n        return {\n            // Dados globais\n            coins: globalStats.coins,\n            xp: globalStats.xp,\n            spinTickets: globalStats.spin_tickets,\n            activeTheme: globalStats.active_theme,\n            \n            // Dados do jogo\n            level: gameStats.level,\n            score: gameStats.score,\n            \n            // Inventário\n            inventory: inventory.map(i => i.item_id)\n        };\n    } catch (error) {\n        console.error('❌ Erro ao carregar dados:', error);\n        return null;\n    }\n}\n\n/**\n * Salvar dados do usuário\n */\nasync function saveUserData(userId, data) {\n    const client = getSupabaseClient();\n    \n    try {\n        // 1. Atualizar dados globais\n        if (data.coins !== undefined || data.xp !== undefined) {\n            await client\n                .from('game_stats')\n                .update({\n                    coins: data.coins,\n                    xp: data.xp,\n                    spin_tickets: data.spinTickets,\n                    active_theme: data.activeTheme\n                })\n                .eq('user_id', userId);\n        }\n        \n        // 2. Atualizar dados específicos do jogo\n        if (data.level !== undefined || data.score !== undefined) {\n            await client\n                .from('novo_jogo_stats')\n                .update({\n                    level: data.level,\n                    score: data.score\n                })\n                .eq('user_id', userId);\n        }\n        \n        console.log('✅ Dados salvos com sucesso');\n    } catch (error) {\n        console.error('❌ Erro ao salvar dados:', error);\n    }\n}\n```\n\n---\n\n## 🎮 Integração de Novo Jogo\n\n### Passo 1: Criar Pasta do
